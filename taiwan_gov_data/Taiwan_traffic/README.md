# Taiwan A1 & A2 Traffic Analysis


## Taiwan Traffic Classification
- A1 :  造成人員當場或二十四小時內死亡之交通事故。
- A2 :  造成人員受傷或超過二十四小時死亡之交通事故。
- A3 :  僅有財物損失之交通事故。
